class opendp.smartnoise.sql.Rewriter(metadata)[source]¶

Bases: object

Modifies parsed ASTs to augment with information needed to support differential privacy. Uses a matching object which contains metadata necessary for differential privacy, such as min/max, cardinality, and key columns.

This routine is intended to be used prior to post-processing with random noise generation.
